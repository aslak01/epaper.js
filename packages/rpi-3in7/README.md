# @epaperjs/rpi-3in7

Node bindings for [Waveshare 3.7"](https://www.waveshare.com/3.7inch-e-paper-hat.htm)

## Supported Color Modes

`Black / White` \
`4 Gray`

## Installation

```
npm i -g @epaperjs/rpi-3in7
```
